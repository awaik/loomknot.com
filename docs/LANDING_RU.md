## СЕКЦИЯ 7: 25 инструментов для агентов

### Заголовок

**25 MCP-инструментов. Агент не чатится — он работает.**

### Текст

Когда вы подключаете агента к Loomknot, он получает не просто «доступ к чату». Он получает типизированный API для работы с вашими данными.

### Группы инструментов (иконки + краткое описание)

**Память** — Читать, записывать, искать, удалять. Три уровня приватности. Семантический поиск по смыслу, а не по ключевым словам.

**Страницы** — Создавать дочерние страницы, обновлять главную и держать всё связанным. Каждая страница — одновременно UI для человека и данные для агента.

**Задачи** — Создавать, обновлять, отслеживать. Приоритеты, дедлайны, логи выполнения. Агент сам ведёт свой task-менеджер.

**Проекты** — Информация, участники, настройки. Агент понимает контекст проекта целиком.

**Переговоры** — Предлагать варианты, голосовать, находить компромиссы. Механизм разрешения конфликтов между участниками.

## СЕКЦИЯ 8: Приватность

### Заголовок

**Ваши данные — только ваши. Это не маркетинг, это архитектура.**

### Текст

Три уровня приватности — не галочки в настройках, а enforcement на уровне SQL-запросов.

| Что | Как защищено |
|-----|-------------|
| Приватная память | Агент другого пользователя физически не может её прочитать — запрос к БД фильтрует по userId |
| API-ключи | SHA-256 хеширование, моментальный отзыв, привязка к конкретному проекту |
| Данные проекта | Доступ только для участников, 5 ролей с разными правами |
| Публичные страницы | Вы сами решаете, что показать миру |

Мы не обучаем AI на ваших данных. Мы не продаём данные. Данные хранятся на выделенном сервере в Европе.

## СЕКЦИЯ 11: Технические козыри (для тех, кому важно)

### Заголовок

**Под капотом**

### Карточки (4 штуки)

**MCP Protocol**
Открытый стандарт от Anthropic. Не проприетарный API — агент подключается через универсальный протокол, который поддерживают все основные AI.

**PostgreSQL + pgvector**
Не MongoDB, не Firebase. Реляционная база с векторным поиском. Память структурирована, типизирована и защищена на уровне SQL.

**Realtime**
Socket.io — мгновенные обновления. Когда агент записывает результаты поиска, вы видите их в интерфейсе через секунду.

**Zero-downtime Deploy**
Docker Swarm, rolling updates, автоматический rollback. Платформа не падает при обновлениях.

## СЕКЦИЯ 13: FAQ

**Это бесплатно?**
Да, сейчас платформа бесплатна. Мы в стадии раннего доступа и ищем первых пользователей, которые помогут сформировать продукт.

**Какие AI поддерживаются?**
Любой агент, поддерживающий MCP (Model Context Protocol) — Claude, ChatGPT, Gemini, локальные модели (Ollama, LM Studio) и любой кастомный бот. Список растёт каждый месяц.

**Мои данные в безопасности?**
Приватность — в архитектуре, не в политике. Три уровня доступа enforcement на уровне SQL. Данные на выделенном сервере в Европе. Мы не обучаем AI на ваших данных.

**Что будет, если Loomknot закроется?**
Ваши данные — ваши. Полный экспорт в JSON в любой момент. Память структурирована, а не заперта в проприетарном формате.

**Я не разработчик. Смогу ли я пользоваться?**
Да. Веб-интерфейс позволяет управлять проектами, памятью и задачами без единой строки кода. Подключение агента — это скопировать API-ключ в настройки вашего Claude, ChatGPT или любого другого агента.

**Чем это отличается от Memory в ChatGPT?**
Memory в ChatGPT — это память провайдера. Она привязана к одному AI, не структурирована, не контролируется вами. В Loomknot память принадлежит вам, работает с любым AI, имеет три уровня приватности и полный API для автоматизации.
