# SEO Optimizer

Комплексная SEO-оптимизация страниц для максимальной видимости в поисковиках (Яндекс, Google).

## Workflow

При запросе на SEO-оптимизацию выполняй следующие шаги:

### 1. Анализ страницы

Прочитай файл страницы и определи:
- Текущий title и description (в metadata/generateMetadata)
- Структуру заголовков (H1, H2, H3)
- Ключевые слова по контенту
- Внутренние ссылки
- Schema markup (если есть)

### 2. SEO-аудит по чеклисту

Проверь каждый пункт и выстави оценку:

#### Title Tag (4 балла)
| Проверка | Балл | Критерий |
|----------|------|----------|
| Длина 50-60 символов | 1 | Не обрезается в выдаче |
| Ключевое слово в начале | 1 | Первые 3-4 слова |
| Уникальный и привлекательный | 1 | Вызывает желание кликнуть |
| Бренд в конце (опционально) | 1 | " - Планета тестов" |

#### Meta Description (4 балла)
| Проверка | Балл | Критерий |
|----------|------|----------|
| Длина 150-160 символов | 1 | Полностью показывается |
| Начинается с глагола действия | 1 | "Пройдите", "Узнайте", "Проверьте" |
| Содержит ключевое слово | 1 | Естественно вписано |
| Call-to-action | 1 | Мотивирует кликнуть |

#### Контент (6 баллов)
| Проверка | Балл | Критерий |
|----------|------|----------|
| H1 с ключевым словом | 1 | Один H1 на странице |
| Ключевое слово в первых 100 словах | 1 | Ранняя релевантность |
| Минимум 300 слов контента | 1 | Достаточная глубина |
| Подзаголовки H2/H3 структурированы | 1 | Логическая иерархия |
| Изображения с alt-текстом | 1 | Описательные alt |
| Внутренние ссылки (3-5 шт) | 1 | На релевантные страницы |

#### Технический SEO (4 балла)
| Проверка | Балл | Критерий |
|----------|------|----------|
| Canonical URL | 1 | Указан правильно |
| Open Graph теги | 1 | og:title, og:description, og:image |
| Schema.org разметка | 1 | Структурированные данные |
| URL содержит ключевое слово | 1 | Читаемый slug |

### 3. Генерация рекомендаций

На основе аудита сгенерируй:

```markdown
## SEO-аудит: [Название страницы]

**Текущий балл:** XX/18
**Статус:** ✅ Отлично (15+) | ⚠️ Нужны улучшения (10-14) | ❌ Критично (<10)

### Текущее состояние
- **Title:** "[текущий title]" (XX символов)
- **Description:** "[текущее описание]" (XX символов)
- **H1:** "[текущий H1]"
- **URL:** /путь/к/странице

### Рекомендации

#### 🔴 Критичные (сделать сейчас)
1. [Конкретная рекомендация]
   - Текущее: ...
   - Рекомендуемое: ...

#### 🟡 Важные (улучшат позиции)
1. [Рекомендация]

#### 🟢 Опционально (бонус)
1. [Рекомендация]
```

### 4. Применение оптимизаций

После одобрения пользователя:

1. **Обнови metadata** в page.tsx:
```tsx
export const metadata: Metadata = {
  title: "Оптимизированный title - Планета тестов",
  description: "Оптимизированное описание 150-160 символов с ключевым словом и призывом к действию.",
  openGraph: {
    title: "Title для соцсетей",
    description: "Описание для соцсетей",
    type: "website",
  },
};
```

2. **Добавь Schema.org** если нет:
```tsx
<script
  type="application/ld+json"
  dangerouslySetInnerHTML={{
    __html: JSON.stringify({
      "@context": "https://schema.org",
      "@type": "WebPage", // или Test, Article, etc.
      "name": "Название",
      "description": "Описание",
    }),
  }}
/>
```

3. **Оптимизируй контент:**
   - Добавь ключевое слово в первый абзац
   - Структурируй с H2/H3
   - Добавь внутренние ссылки

## Шаблоны для Планета тестов.ru

### Страница теста
```
Title: [Название теста] - Бесплатный онлайн тест | Планета тестов (55-60 символов)
Description: Пройдите [название теста] онлайн бесплатно. [Что измеряет]. [Количество] вопросов, результат сразу. Научная методика [автор/год].
```

**Пример:**
- Title: "Тест Бека на депрессию - Бесплатно онлайн | Планета тестов" (54 символа)
- Description: "Пройдите тест Бека на депрессию онлайн бесплатно. Определите уровень депрессии по шкале BDI-II. 21 вопрос, результат сразу. Научная методика с 1961 года." (156 символов)

### Страница категории
```
Title: [Категория] тесты онлайн бесплатно | Планета тестов
Description: Пройдите лучшие тесты на [категорию] онлайн. [X] научных тестов с мгновенным результатом. Проверьте [что проверяют] бесплатно.
```

### Страница тега
```
Title: Тесты на [тег] - Онлайн проверка бесплатно | Планета тестов
Description: [X] тестов на [тег]. Проверьте [что проверяют] с помощью научных методик. Бесплатно, анонимно, результат сразу.
```

## Schema.org для тестов

### Тип Quiz (для тестов)
```json
{
  "@context": "https://schema.org",
  "@type": "Quiz",
  "name": "Название теста",
  "description": "Описание теста",
  "educationalLevel": "beginner",
  "about": {
    "@type": "Thing",
    "name": "Психология"
  },
  "provider": {
    "@type": "Organization",
    "name": "Планета тестов",
    "url": "https://testplanet.ru"
  }
}
```

### BreadcrumbList (хлебные крошки)
```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Тесты",
      "item": "https://testplanet.ru/tests"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Тревожность",
      "item": "https://testplanet.ru/tests/categories/anxiety"
    }
  ]
}
```

## Featured Snippets (быстрые ответы)

Для попадания в featured snippets добавляй структурированные ответы:

### Формат "Что такое"
```markdown
## Что такое тест Бека на депрессию?

**Тест Бека на депрессию (BDI-II)** — это научный опросник из 21 вопроса для оценки уровня депрессии. Разработан Аароном Беком в 1961 году. Тест измеряет когнитивные, эмоциональные и физические симптомы депрессии.
```
(40-60 слов, ключевое слово в начале, жирным)

### Формат списка
```markdown
## Как пройти тест?

1. Прочитайте вопрос внимательно
2. Выберите вариант, который лучше описывает ваше состояние
3. Отвечайте честно, результаты анонимны
4. После всех вопросов получите результат
```

### Формат таблицы (для сравнений)
```markdown
## Интерпретация результатов

| Баллы | Уровень | Рекомендация |
|-------|---------|--------------|
| 0-9 | Норма | Профилактика |
| 10-18 | Лёгкая | Самопомощь |
| 19-29 | Умеренная | Консультация |
| 30+ | Тяжёлая | Обратиться к врачу |
```
